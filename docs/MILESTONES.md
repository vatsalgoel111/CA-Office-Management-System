# Milestone Plan

## Milestone 1: Project Foundation

Done when:

- Folder structure exists
- Python environment is configured
- Logging works
- SQLite connection works
- Database initialization can run
- Tests can run
- Git commit is created

## Milestone 2: Login and Roles

Done when:

- Administrator can log in
- Staff can log in
- Invalid login is rejected
- Role-based dashboard opens
- Tests pass
- Git commit is created

## Milestone 3: Client Management

Done when:

- Clients can be added
- Clients can be edited
- Clients can be searched
- Clients can be deactivated
- Client data persists in SQLite
- Tests pass
- Git commit is created

## Milestone 4: Work Allocation

Done when:

- Administrator assigns work
- Staff sees assigned work
- Staff updates status
- Remarks are stored
- Tests pass
- Git commit is created

## Milestone 5: Billing and Collections

Done when:

- Bills can be created
- Payments can be recorded
- Outstanding amount is calculated
- Tests pass
- Git commit is created

## Milestone 6: Reports and Exports

Done when:

- Reports can be filtered
- Excel export works
- PDF export works
- Tests pass
- Git commit is created

## Milestone 7: Automation

Done when:

- Audit log works
- Automatic backup works
- Reminders work
- WhatsApp notification proof of concept works
- Tests pass
- Git commit is created

## Milestone 8: Packaging

Done when:

- Executable is created
- App runs on a clean Windows system
- Production paths are correct
- Deployment checklist is complete
- Git commit is created
